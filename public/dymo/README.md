# DYMO LabelWriter Setup

DYMO Connect Framework is installed in this directory.

## Current Status

✅ SDK files are present:
- `dymo-connect-framework-master/dymo.connect.framework.js`
- `dymo-connect-framework-master/dymo.connect.framework.full.js`

## Remaining Setup

1. **Install DYMO LabelWriter software:**
   - Download DYMO Connect software from: https://www.dymo.com/dymo-label-software
   - Install it on your computer
   - Connect your DYMO LabelWriter 550

## Ready to Use

The DYMO integration is now configured. Once you:
1. Install DYMO Connect software
2. Connect your DYMO LabelWriter 550

The label printing will be fully functional:
- "Print Label" button in order detail modal
- Customizable label templates at /admin/settings/labels
- Labels will print to your connected DYMO LabelWriter 550

## Troubleshooting

If label printing doesn't work:
1. Ensure DYMO LabelWriter is connected and powered on
2. Ensure DYMO Connect software is installed
3. Check browser console for error messages
4. Ensure you're using Chrome or Edge (best browser support for DYMO)
